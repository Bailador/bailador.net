Bailador

Web Application Development in Perl 6

Bailador is a light-weight, route-based, next-generation, fully buzzword compatible framework for web application development. The goals of Bailador include:

  • Provide an easy way to develop Microservices that only communicate with APIs
  • Easily get started to build a content-heavy web site that can progress to a more dynamic application.
  • Batteries included: We would like to let our users focus on their problems and not spend time picking the "right JSON parser".
  • Flexibility: On the other hand if they really need their own JSON parser we would like to make sure they can use it.
  • Accessibility: We would like to make it easy to create accessible web applications. We know. We have a long way to go.
